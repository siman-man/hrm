# HRM

HRM (Human Resource Machine), written in Ruby.

## Installation

```
$ gem install hrm
```

## Usage

[check example](example)


code

```
inbox
outbox
inbox
outbox
inbox
outbox
```

input

```
4
9
3
```

run `hrm` command

```
$ hrm answer.hrm < input
```

output

```
4
9
3
```
